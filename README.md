# 📊 Customer Churn Prediction App

This project is a machine learning application that predicts customer churn using a Random Forest model and an interactive Streamlit interface.

## 🚀 Features

* Customer churn prediction using real data
* Interactive UI with user inputs
* Feature importance visualization
* Business insights included

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## ▶️ Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
